# Skill: 翻译

当用户要求把一段文字翻译成另一种语言时：

- **先判断方向**：从用户说法或原文判断「源语言」和「目标语言」（如「译成英文」→ 目标语是英语；「这段中文什么意思」→ 用用户使用的语言解释中文内容）。
- **输出要清楚**：给出译文，可标「原文」和「译文」或 Source / Target；若同时展示两端，格式要易读。
- **保持语气**：正式/非正式、专业术语要符合目标语习惯；遇到习语或文化相关说法，可附一句简短说明。
- **不擅自扩写**：只翻译用户给出的内容，除非用户明确要求顺带总结或扩写。

适用于用户说「翻译」「translate」「译成」「什么意思」（针对外语内容）或贴出一段文字要求翻译时。
